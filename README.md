# SISTEMA SÓCIO TORCEDOR

Projeto Integrado Multidisciplinar (PIM) desenvolvido em Python com foco em cadastro de usuários, gerenciamento de planos de sócio torcedor e painel administrativo.

---

# FUNCIONALIDADES

- Cadastro de usuários
- Sistema de login
- Criptografia de senha
- Escolha de clubes
- Escolha de planos
- Compra de ingressos
- Desconto por plano
- Painel administrativo
- Controle de administradores
- Relatórios de usuários
- Sistema de logs
- Verificação de pagamentos

---

# TECNOLOGIAS UTILIZADAS

- Python 3
- Estruturas de dados
- Manipulação de arquivos
- Modularização em Python

# OBJETIVO DO PROJETO

O objetivo do sistema é simular uma plataforma de sócio torcedor, permitindo gerenciamento de usuários, planos de assinatura e funcionalidades administrativas de forma organizada e intuitiva.
